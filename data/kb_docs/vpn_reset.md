# VPN Password Reset Process

To reset your VPN password:

1. Go to the company password portal.
2. Sign in using your employee ID.
3. Select "Reset VPN Password".
4. Complete MFA verification.
5. Create a new password following the password policy.
6. Wait 5 minutes before reconnecting to VPN.

If the reset does not work, raise a ServiceNow incident with category "Network/VPN".
